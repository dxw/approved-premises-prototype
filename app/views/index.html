Apply

From the dashboard, a user can select a case that is in progress, or can start a new case. If the user starts a new case, they will be taken to the start screen. If a user selects a case that is in progress, they will be taken to the task list.

If the person's tier does not match the elegibility requirements, users will be shown the eligibility screen where they can provide additional information for exceptional circumstances.

  • If the user selects yes, they will be shown exemption questions screen
  • If the user selects no, they will be shown a drop out screen

Multi-agency & emergency screens

We need to show additional questions for applications that are Post sentence supervision or Non-Statutory. PP's will need to provide additional information to form a contingency plan.

The following screens will be shown to users within the further placement considerations task

  • Input multi-agency partners
  • Input multi-agency question responses

If an application is an emergency case, the PP will need to provide follow up information for the AP manager. They will be shown the following screens at the beginning of the 'Further placement considerations' task flow. Contingency plans - emergency

Emergency applications

Emergency applications are for applications which are less than 7 days. If the application is submitted in less than four months an additional screen will be presented.

The pages will be presented if the arrival date is either less than 7 days, or 4 months of the application date.

If the screen is less than 4 months, or less than 7 days they will be shown the following screen

Request for placement screens

If an application is parole (where no date of release is confirmed) or a ROTL, the probation practicioner will need to create a request for placement.

Assess

An assessor will log in and arrive at their dashboard where they can view cases that are in progress, have requested further information from a PP and cases that have been submitted.

If a user wants to view a case that's in progress, they will be taken to a task list where they can work through each task.
