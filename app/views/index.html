Apply

From the dashboard, a user can select a case that is in progress, or can start a new case. If the user starts a new case, they will be taken to the start screen. If a user selects a case that is in progress, they will be taken to the task list.

Emergancy referrals

Emergancy referrals are for applications which are less than 4 months. These can be submitted by a practicioner in a standard way. If the release date is less than four months, the user will be presented with the 'emergancy application tasklist'.

There are two additional tasks to complete.

Assess

An assessor will log in and arrive at their dashboard where they can view cases that are in progress, have requested further information from a PP and cases that have been submitted.

If a user wants to view a case that's in progress, they will be taken to a task list where they can work through each task.

Additional assessment questions (PIPE, ESAP, Emergancy)

For emergancy, PIPE and ESAP assessments, users will be presented with an additional task on the list. They will be required to enter extra information to assess the information that's been provided in the Application.
