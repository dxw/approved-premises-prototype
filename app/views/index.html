Apply

From the dashboard, a user can select a case that is in progress, or can start a new case. If the user starts a new case, they will be taken to the start screen. If a user selects a case that is in progress, they will be taken to the task list.

If the person's tier does not match the elegibility requirements, users will be shown the eligibility screen where they can provide additional information for exceptional circumstances.

  • If the user selects yes, they will be shown exemption questions screen
  • If the user selects no, they will be shown a drop out screen

Design history

Screen Date of change Changes made Reason for change
Import OASys screen 12 Oct 2022 We've moved 'Drug and Alcohol' from Needs linked to risk of serious harm are automatically imported to the section above, "We'll automatically import:" We have agreed that these will be imported for all users, so would like to show that this is imported with the other information that's automatically imported.
Risk information - ROSH 12 Oct 2022 We've removed the free text box at the bottom of the page for users to enter 'How will AP placement support this area in management of risk' Users already provide detail about risk management on the 'purpose of placement' and 'Useful features of an AP' screen. We have not tested this with users, but believe the textbox might be confusing for users, as it's an empty textbox at the bottom of the OASys information. There's an assumption that this may be missed, or mistaken as an empty OASys field that's been pulled in. We're going to remove it for now and see how this tests in research.
