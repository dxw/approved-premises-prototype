Multi-agency & emergency screens

Some applications for PSS (Post sentence supervision) and Non-statutory cases require contingency plans. These are used in the Approved Premises for out of hours assistance.

If the type of application is either of the above, the following questions will be shown under 'further placement considerations' task

If an application is an emergency case, the PP will only need to provide information about the multi-agency questions

  • Input multi-agency partners
  • Input multi-agency question responses

Emergency applications

Some applications will be emergency or short term applications.

  • Emergency applications are those submitted where arrival is less than 7 days.
  • Short notice applications are those within a 28 day period.

Users will be asked for the reason for a short term application.

Request for placement screens

Once a parole or ROTL application has been assessed (as suitable), PPs can submit a request for placement. This is done when a date of arrival is known.
